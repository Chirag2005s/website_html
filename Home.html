HTML web Development

HTML whose full name is Hypertext Markup Language, it is a language of the computer which is used to create the webpage. Simply put, HTML is a programming language with the help of which new webpages can be created. All the webpages we see in our web browser are made in HTML itself.

CSS web Development

HTML and CSS is a computer language which is very simple and can be taught easily. To write the code of html and css, we need a text editor such as Notepad. After writing these codes, a web browser is needed to view it through the internet.

JavaScript web Development

JavaScript is a popular computer programming language, with the help of which HTML web pages can be made more interactive, it is a powerful scripting language. JavaScript supports almost all web browsers and operating systems.
